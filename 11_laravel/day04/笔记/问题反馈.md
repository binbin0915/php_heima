##  能够掌握Laravel中session数据的存储
```
session([])   # 设置

session(‘’)  # 获取

session()->has() # 判断是否存在

session()->flush() # 清除所有

session()->flash(key,value)  # 闪存

redirect()->route()->with(key,value) # 闪存
```
